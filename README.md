# 🔢 2048 Game

> 🎮 Game 2048 chạy trực tiếp trên trình duyệt – đơn giản nhưng cực kỳ gây nghiện và mang tính chiến thuật cao.

🌐 **Live Demo:** https://panbap.github.io/2048/

---

## 🚀 Giới thiệu

Đây là một project **2048 Game** được xây dựng bằng HTML, CSS và JavaScript.

Người chơi di chuyển các ô số trên bảng 4x4 để ghép các số giống nhau, tạo ra các giá trị lớn hơn và đạt mục tiêu **2048**. :contentReference[oaicite:0]{index=0}  

---

## 🎮 Gameplay

- Sử dụng phím mũi tên (↑ ↓ ← →) hoặc swipe để di chuyển  
- Khi 2 ô giống nhau chạm nhau → sẽ gộp lại  
- Mỗi lần gộp → điểm tăng  
- Game kết thúc khi không còn nước đi
